Q1: "Bagaimana jika ada kesalahan entry data?"
A: "Aplikasi punya validasi untuk input. Jika ada data yang salah, bisa langsung 
di-edit melalui menu Edit di setiap halaman. Semua perubahan dicatat di database."

Q2: "Apakah bisa backup data?"
A: "Ya, database MySQL bisa di-backup melalui phpMyAdmin atau command line MySQL. 
Backup bisa dijadwalkan secara otomatis setiap hari."

Q3: "Berapa pengguna yang bisa akses bersamaan?"
A: "Aplikasi bisa handle multiple users. Username & password bisa dibuat untuk 
berbagai operator yang berbeda dengan role-based access (Admin, Kasir, dll)."

Q4: "Bagaimana jika internet down?"
A: "Aplikasi ini lokal (tidak memerlukan internet). Bisa akses bahkan tanpa 
internet selama komputer server tetap menyala dan database berfungsi."

Q5: "Berapa lama waktu training untuk operator?"
A: "Sangat mudah. Interface intuitif dan menu navigasi jelas. Operator baru bisa 
terlatih dalam 1-2 jam dengan guidance."

Q6: "Bisa di-customize sesuai kebutuhan?"
A: "Ya, source code terbuka dan bisa di-modify. Untuk fitur custom, bisa request 
development tim."

Q7: "Bagaimana dengan laporan bulanan/tahunan?"
A: "Fitur export Excel memudahkan membuat laporan. Data bisa langsung di-analyze 
di Excel. Untuk laporan automated, sedang dalam development."

Q8: "Apakah ada password recovery jika lupa password?"
